# KeepNotes

Keep Note is simple web applications to store your notes in a managed way.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install its dependencies in **keepnotes**.

```bash
cd keepnotes
yarn
yarn dev # for dev
yarn start # for dev
```
